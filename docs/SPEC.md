# ポケモン ダメージ計算ツール 仕様書

## 1. 概要
JavaScriptを使用した、ポケモンのダメージ計算シミュレーター。
攻撃側のステータスと防御側のステータスを入力し、与えるダメージの範囲を表示する。

## 2. 画面レイアウト案 (ASCII Layout)
```text
+-------------------------------------------------------+
|              ポケモン ダメージ計算ツール              |
+-------------------------------------------------------+
| [ 攻撃側設定 ]                  [ 防御側設定 ]        |
|  攻撃力: [ 150 ]                 HP:     [ 175 ]      |
|  技威力: [ 90 ]                  防御力: [ 100 ]      |
|  タイプ相性: [ 1.0 ▼]                                |
+-------------------------------------------------------+
|                                                       |
|   < 計算実行ボタン >                                  |
|                                                       |
+-------------------------------------------------------+
| [ 計算結果 ]                                          |
|  ダメージ: 120 ～ 142                                 |
|  割合: 68.5% ～ 81.1% (確定2発)                       |
+-------------------------------------------------------+
```

## 3. 主要な機能・項目

### 入力項目

- 攻撃側
  - 攻撃実数値（または特攻）
  - 技の威力
  - タイプ相性倍率（x4, x2, x1, x0.5, x0.25, x0）
- 防御側
  - HP実数値
  - 防御実数値（または特防）

### 計算ロジック

- 基本ダメージ式: ((50 * 2 / 5 + 2) * 威力 * 攻撃 / 防御) / 50 + 2
- 乱数補正: 0.85 ～ 1.00 の範囲で計算
