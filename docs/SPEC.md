# ポケモン ダメージ計算ツール 仕様書

## 1. 概要
JavaScriptを使用した、ポケモンのダメージ計算シミュレーター。
攻撃側のステータスと防御側のステータスを入力し、与えるダメージを表示する。

## 5. 補足

### ダメージ計算の前提条件

チャンピオンズのルールを想定して以下を前提条件とする。

- レベルは50で固定
- 個体値は31で固定
- 計算時、明記されていない場合は小数点以下を発生する都度切り捨てる

#### ステータス計算

- HP
  - 種族値 + 75 + 能力ポイント
- こうげき、ぼうぎょ、とくこう、とくぼう
  - (種族値 + 20 + 能力ポイント) * 性格補正

#### ダメージ計算

- 基礎ダメージ
  - (22 * 技威力 * こうげき / ぼうぎょ) / 50 + 2
- タイプ一致補正 (攻撃側設定のポケモンのタイプと技のタイプが一致している場合)
  - 基礎ダメージ * 1.5
- タイプ相性
  - 基礎ダメージ * タイプ相性
- 乱数補正
  - 0.85 ～ 1.00 の範囲で計算
  - ダメージに乱数補正の"最小～最大"を表示
  - 割合に乱数補正の最小/最大のHP割合を"最小～最大"で表示
    - 小数点以下1桁を切り捨てる
