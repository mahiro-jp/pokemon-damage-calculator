# ポケモン ダメージ計算ツール 仕様書

## 1. 概要
JavaScriptを使用した、ポケモンのダメージ計算シミュレーター。
攻撃側のステータスと防御側のステータスを入力し、与えるダメージの範囲を表示する。

## 2. 画面レイアウト案 (ASCII Layout)
```text
+------------------------------------------------------------------+
|                    ポケモン ダメージ計算ツール                   |
+------------------------------------------------------------------+
|  [ 攻撃側設定 ]                  [ 防御側設定 ]                  |
|  ポケモン名:     [ サザンドラ ]  ポケモン名:     [ リザードン ]  |
|  性格補正:           [ x1.1 ▼]  性格補正:             [ x1 ▼]  |
|  能力ポイント:           [ 32 ]  HP能力ポイント:         [ 32 ]  |
|  攻撃実数値:            [ 150 ]  HP:                    [ 175 ]  |
|  技名:         [ あくのはどう ]  防御能力ポイント:       [ 32 ]  |
|  技威力:                 [ 80 ]  防御実数値:            [ 100 ]  |
|  タイプ相性:          [ 1.0 ▼]                                  |
+------------------------------------------------------------------+
|                                                                  |
|                       < 計算実行ボタン >                         |
|                                                                  |
+------------------------------------------------------------------+
|                           [ 計算結果 ]                           |
|                             ダメージ:                            |
|                            120 ～ 142                            |
|                  割合: 68.5% ～ 81.1% (確定2発)                  |
+------------------------------------------------------------------+
```

## 3. 画面項目

- 攻撃側設定
  - ポケモン名
    - 手入力、リストから選択
  - 性格補正
    - 選択肢 (x1.1, x1, x0.9)
  - 能力ポイント
    - 手入力、入力範囲 (0 ~ 32)
  - 攻撃実数値 (自動計算)
  - 技威力 (自動計算)
  - タイプ相性
    - 選択肢 (x4, x2, x1, x0.5, x0.25, x0)
- 防御側
  - ポケモン名
    - 手入力、リストから選択
  - 性格補正
    - 選択肢 (x1.1, x1, x0.9)
  - HP能力ポイント
    - 手入力、入力範囲 (0 ~ 32)
  - HP実数値 (自動計算)
  - 防御能力ポイント
    - 手入力、入力範囲 (0 ~ 32)
  - 防御実数値 (自動計算)
- 計算実行ボタン
- 計算結果
  - ダメージ (自動計算)
  - 割合 (自動計算)

## 4. アクション

- 攻撃側設定
  - ポケモン名
    - 値を変更する
      - 性格補正、能力ポイント、技名、技威力、タイプ相性を初期化
      - 攻撃実数値を再計算
    - 値をクリアする
      - 性格補正、能力ポイント、攻撃実数値、技名、技威力、タイプ相性を初期化
  - 性格補正
    - 値を変更する
      - 攻撃実数値を再計算
  - 能力ポイント
    - 値を変更する
      - 攻撃実数値を再計算
    - 値をクリアする
      - 攻撃実数値に0を設定
  - 技名
    - 値を変更する
      - 変更した技に合わせて技威力を更新
      - タイプ相性を初期化
    - 値をクリアする
      - 技威力をクリア
      - タイプ相性を初期化
- 防御側設定
  - ポケモン名
    - 値を変更する
      - 性格補正、HP能力ポイント、防御能力ポイントを初期化
      - HP実数値、防御実数値を再計算
    - 値をクリアする
      - 性格補正、HP能力ポイント、HP実数値、防御能力ポイント、防御実数値を初期化
  - 性格補正
    - 値を変更する
      - 防御実数値を再計算
  - HP能力ポイント
    - 値を変更する
      - HP実数値を再計算
    - 値をクリアする
      - HP実数値に0を設定
  - 防御能力ポイント
    - 値を変更する
      - 防御実数値を再計算
    - 値をクリアする
      - 防御実数値に0を設定
- 計算実行
  - クリックする
    - 入力された内容からダメージ計算を行う（詳細は計算ロジックを参照）

### 計算ロジック
レベル50固定、個体値31固定で計算する。

#### ステータス計算
- HP実数値
  - 種族値 + 75 + 能力ポイント
- 攻撃、防御の実数値
  - 種族値 + 20 + 能力ポイント * 性格補正

#### ダメージ計算
- ダメージ
  - (22 * 威力 * 攻撃実数値 / 防御実数値) / 50 + 2
- タイプ一致補正 (攻撃側設定のポケモンのタイプと技のタイプが一致している場合)
  - ダメージ * 1.5
- タイプ相性
  - ダメージ * タイプ相性
- 乱数補正
  - 0.85 ～ 1.00 の範囲で計算
