# ポケモン ダメージ計算ツール 仕様書

## 1. 概要
JavaScriptを使用した、ポケモンのダメージ計算シミュレーター。
攻撃側のステータスと防御側のステータスを入力し、与えるダメージの範囲を表示する。

## 3. 主要な機能・項目

### 入力項目

- 攻撃側
  - ポケモン名
  - 性格補正 (x1.1, x1, x0.9)
  - 能力ポイント (0 ~ 32)
  - 攻撃実数値 (自動計算)
  - 技威力
  - タイプ相性倍率（x4, x2, x1, x0.5, x0.25, x0）
- 防御側
  - ポケモン名
  - 性格補正 (x1.1, x1, x0.9)
  - 能力ポイント (0 ~ 32)
  - HP
  - 防御実数値 (自動計算)

### 計算ロジック
レベル50固定、個体値31固定で計算する。

#### ステータス計算
- HP実数値
$$HP = \text{種族値} + 75 + 能力ポイント$$
- その他
$$\text{実数値} = (\text{種族値} + 20 + 能力ポイント) \times \text{性格補正}$$

#### ダメージ計算
- 基本ダメージ式: (22 * 威力 * 攻撃 / 防御) / 50 + 2
- 乱数補正: 0.85 ～ 1.00 の範囲で計算
